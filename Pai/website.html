核心功能

为高效笔记而生

插件化格式系统

引擎与格式完全解耦。Markdown、思维导图、Drawio、任意格式 — 通过安装插件扩展，无需修改核心代码。

桌面置顶

任何笔记窗口一键 Pin 到桌面最顶层。自定义透明度、贴边自动收起、多窗口同时置顶。

AI 智能生成

自然语言生成笔记内容，自动匹配当前格式。上传文件、图片、语音自动转为结构化笔记。

文件管理

VSCode 风格文件树。新建文档与目录、拖拽排序、右键菜单。拖拽外部文件自动在新窗口打开。

多窗口独立

每个笔记可独立打开为新窗口，自由拖拽到任意显示器。窗口间完全独立，互不干扰。

所见即所得

内置 Milkdown WYSIWYG 编辑器。实时渲染 Markdown，编辑即预览，支持 GFM 语法。

插件系统

你的格式，你来定义

// 定义插件
import { definePlugin } from 'painote-plugin-sdk'

export default definePlugin({
  manifest: {
    id: 'mindmap',
    name: 'Mind Map',
    type: 'format',
    extensions: ['.mm']
  },

  onActivate(ctx) {
    ctx.logger.info('Ready!')
  },

  format: {
    render(content) { /* ... */ },
    getDefaultContent() {
      return '# Central Topic\n'
    }
  }
})

标准 SDK，无限扩展

PaiNote 提供完整的插件 SDK，定义了清晰的生命周期、格式接口和 AI 适配器。第三方开发者可以在几分钟内创建自己的笔记格式插件。

  • 完整的生命周期管理：安装、激活、停用、卸载
  • 格式适配器：每种格式定义自己的渲染和编辑逻辑
  • AI 集成：插件可定义专属 AI prompt 模板
  • 插件商城：可视化安装、启用、卸载插件

笔记浮在桌面之上
